# AI-Powered Fake News Detection

## Overview
This project uses Machine Learning and NLP techniques to classify news articles as Fake or Real.

## Features
- Fake News Detection
- NLP Text Processing
- TF-IDF Vectorization
- Multiple ML Models
- Hindi Language Support

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
